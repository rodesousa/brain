# Facet : examples-ecosystem

Le repo fournit des starters par framework frontend :

- `examples/next`
- `examples/react`
- `examples/svelte`
- `examples/vue`

Chacun contient un `app.zon`, un shell Zig minimal, et un projet frontend.

Pour le dev framework lui-même, voir `CONTRIBUTING.md`.

## Raison du discard

Liste de starters classique pour un framework UI — apporte zero signal différenciant. Le fait qu'il y ait des starters Next/React/Svelte/Vue est attendu pour un framework Vercel Labs et n'éclaire rien sur le design ou le positionnement. À noter ailleurs uniquement si je veux essayer le framework (info opérationnelle, pas conceptuelle).
